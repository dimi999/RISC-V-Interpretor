# RISC-V-Interpretor
Installation
```
git clone https://github.com/dimi999/RISC-V-Interpretor

cd RISC-V-Interpretor
```

Usage
```
python main.py
```

# Error handling
If an instruction is not found in the given tests, it will not work since it is not implemented.
